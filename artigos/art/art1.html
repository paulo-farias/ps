Acessar o gopher com o lynx no termux


Antes de existir a web para conectar-se a internet nos anos 90 era utilizado o gopher, um protocolo diferente do http que serve informações em texto puro. Ainda hoje ele serve a usuários que buscam simplicidade no uso de ferramentas da internet.

Para navegar no gopher é necessário um cliente. Neste artigo vamos lidar com o lynx.

Requisitos:

  • android 5.0 - 12.0
  • f-droid
  • termux
  • termux

Instalação:

Com um celular rodando um sistema android, é necessário o f-droid instalado. O f-droid é um catálogo de aplicativos, funciona como a google play, porém os aplicativos contidos na plataforma são de software livre. No site do f-droid baixe o apk e prossiga com a instalação.

Dentro do aplicativo f-droid, aguarde a atualização do repositório. É automático.

O passo seguinte é instalar o termux, um emulador de terminal para android que executa comandos de unix e pacotes disponíveis do seu próprio repositório.

Após a atualização do repositório no f-droid, procure o app termux no botão pesquisa, vai surgir uma lista de app termux, instale o que aparecer a descrição ‘Emulador de terminal com pacotes’. Durante a data desta postagem, a versão atual do termux é 0.118.0.


- ver captura de imagem no gitlab

O f-droid não recomenda instalar o emulador na google play por motivo do termux não esta recebendo mais atualizações na loja.

Abra o aplicativo termux, e para saber se tudo está funcionando corretamente, atualize o repositório no console: 

				
 pkg update
				
			

				
 pkg upgrade

- ver captura de imagem no gitlab

Por último, dentro do termux, instale o lynx, este programa irá acessar a web em modo texto. Ele é rápido e simples de usar, com suporte para navegação via FTP, Gopher, HTTP, HTTPS, NNTP e o sistema de arquivos local. 

				
 pkg install lynx

Para a execução do lynx é só informar o nome do programa e o parâmetro, uma URL.

				
 lynx gopher://vaporhole.xyz/1/~paulofarias

O controle de navegação entre as páginas são os botões de seta para esquerda(voltar), direita(seguir). Para acessar os menus e rolagem, segue a mesma lógica, seta para baixo(descer)e seta para cima(subir). Botão enter para entrar em um diretório ou ler um arquivo.

Como pesquisar no gopher

Dentro da rede gopher existe sistemas de busca para encontrar outros sites gopher e arquivos, estes sistemas são chamados de índices pesquisáveis, um deles é o Veronica-2.

Encontre o ‘(?) Search Veronica-2’ e pesquise todos os títulos conhecidos na rede.

				
 lynx gopher://floodgap.com

Com acesso a rede no gopherhole floodgap.com, uma listagem de gopher server aparecerá com vários “buracos” DIR(diretórios), documentos e arquivos.
