Termux e conexão no vaporHole


Para quem mora no terminal realizando tarefas de estudos, criando um servidorzinho e codando, chega um momento que precisamos logar no irc ou aparece uma luz para resolver aquele bug na pressa e não é toda hora que temos acesso a um pc.

Meu objetivo com esse tutorial é mostrar alguns comandos de termux para quem é membro do vaporHole ter mais acessibilidade em conectar ao projeto de qualquer lugar.

O termux permite executar comandos e aplicações do linux dentro do emulador. Funcionando como um pequeno sistema operacional, com um total de 2385 pacotes recebendo um trabalho contínuo de atualização. termux-packages

Veja o primeiro artigo na minha página falando da instalação do termux e o uso com o lynx.

Configuração

Tendo o termux instalado e atualizado, para ter acesso remoto de dentro do terminal, é importante uma pré-configuração no console para o controle do gerenciamento de dados entre o termux e o usuário, necessário para a transferência da chave ssh nos documentos do usuário do android até a pasta destino no termux.

digite o comando para ativar o compartilhamento de arquivo:

				
 termux-setup-storage

Conceda as permissões se aparecerem caixas de diálogos. Este comando vai criar um diretório com o nome storage na sua /home e com isso o acesso ao compartilhamento de arquivos já está ativo.

Caso o comando não funcione, é porque o pacote termux-tools não está instalado. Este pacote de ferramentas faz parte do repositório main do Termux Packages e vai resolver o problema. 

				
 Pkg install termux-tools

e depois repita o comando:

				
 termux-setup-storage

- ver captura de imagem no gitlab

Conectando ao vaporHole

É através do utilitário openssh que o termux irá fazer a conexão. O pacote precisa ser baixado. Após baixado, o diretório oculto .ssh é criado na /home do usuário, e é nesse diretório oculto que será transferido sua id_rsa. 

				
 pkg install openssh

Use meios para transferência de arquivo do seu computador para sua pasta dentro do android. Atenção! Utilize uma pasta que o termux fez a linkagem, exemplos das pastas comuns compartilhadas do diretório /storage criado pelo comando termux-setup-storage são: dcim, music, downloads, movies, pictures. E então copie e transfira a chave id_rsa para o diretório /home/.ssh

Com sua chave já dentro do diretório, siga o processo padrão para entrar no servidor como normalmente você entra pelo pc.

Tudo está pronto para aproveitar o vaporHole no seu celular, aprendendo com o fórum interno com dicas de tecnologia, linguagens a seu serviço para programar em C, python, javascript etc. Além de jogos, uma rádio local e outros.
